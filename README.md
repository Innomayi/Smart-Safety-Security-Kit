# Smart Safety & Security Kit

## Overview
The **Smart Safety & Security Kit** is a collection of beginner-friendly Arduino projects designed to demonstrate safety, monitoring, and automation concepts. These projects help learners understand how sensors can be used to detect fire, gas leaks, motion, and sound-based commands.

Each project is provided in a separate folder with its own Arduino code, circuit connections, and documentation.

---

## Projects Included

### 1. Fire Alarm System
**Description:**
Detects the presence of fire or flame using a flame sensor and activates an alarm.

**Components Used:**
- Arduino UNO
- Flame Sensor
- Buzzer
- LED
- Jumper Wires
- Breadboard

**Features:**
- Real-time flame detection
- Audible alarm using buzzer
- Visual indication using LED

---

### 2. Gas Leakage Detection System
**Description:**
Monitors the surrounding environment for gas leakage using an MQ gas sensor and triggers an alarm when gas concentration exceeds a threshold.

**Components Used:**
- Arduino UNO
- MQ Gas Sensor
- Buzzer
- LED
- Jumper Wires
- Breadboard

**Features:**
- Gas leak detection
- Instant audio alert
- Visual warning indication

---

### 3. Motion Detection Alarm
**Description:**
Uses a PIR sensor to detect human movement and activates an alarm system.

**Components Used:**
- Arduino UNO
- PIR Motion Sensor
- Buzzer
- LED
- Jumper Wires
- Breadboard

**Features:**
- Human motion detection
- Security alarm functionality
- LED status indication

---

### 4. Clap Controlled LED
**Description:**
Uses a sound sensor to detect claps and control an LED.

**Components Used:**
- Arduino UNO
- Sound Sensor
- LED
- Resistor
- Jumper Wires
- Breadboard

**Features:**
- Clap detection
- Hands-free LED control
- Introduction to sound-based automation

- Arduino IDE

- USB Cable for Arduino Programming

---

## How to Use

1. Download or clone this repository.
2. Open the desired project folder.
3. Connect the components as shown in the circuit diagram.
4. Open the corresponding `.ino` file in Arduino IDE.
5. Select **Arduino UNO** as the board.
6. Select the correct COM port.
7. Upload the code to the Arduino.
8. Test the project.
